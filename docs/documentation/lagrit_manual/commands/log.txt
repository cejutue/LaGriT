> **_LOG_**

>

>> Turn the batch output file and tty output file **off** and **on**. The tty
prints to and reads from the user's screen. The batch file is the output file
called outx3dgen. Default is **on** for both files.

> FORMAT:

>

>> **log**/**bat**|**tty**/**on**|**off**/

>

> EXAMPLE:

>

>> **log**/**tty**/**off**
