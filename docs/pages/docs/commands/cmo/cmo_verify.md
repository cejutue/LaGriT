# cmo/verify

------------

Verify that Mesh Object is consistent.

## SYNTAX
 
<pre>
<b>cmo/verify</b>/ mo_name /
</pre>

`mo_name` is the mesh object, default is **-all-**.


 
## EXAMPLES

```
cmo/verify/ mo_tet2

cmo/verify/-cmo-

cmo/verify

cmo/verify/-all-

cmo/verify/-default-
```
