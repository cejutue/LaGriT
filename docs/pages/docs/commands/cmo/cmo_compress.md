# cmo/compress

Shortens all memory managed arrays for Mesh Object to their actual lengths.

## SYNTAX

<pre>
cmo/compress/ mo_name
</pre>

`mo_name` is type character, default is the current mesh object.


## EXAMPLES

```
cmo/compress/ mo_tet2

cmo/compress/-cmo-

cmo/compress

cmo/compress/-all-
```
